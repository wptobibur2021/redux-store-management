# Redux Store Management React App

## How to run locally?

### Steps Below
  1.  Copy this command `git clone https://github.com/wptobibur2021/redux-store-management.git` and where you set up the project open cmd past that command.
  2.  Open the redux-store-management folder.
  3.  Open your computer cmd run this command `npm install or yarn install` then run `npm start or yarn start` this command on cmd.  

## How to deploy on netlify?

### Steps Below
  1. Open your project folder, open cmd run this command `npm run build` then create a build folder. 
  2. Go to the netlify website (Must be login) click on the `Sites` menu then you can see the `Add new site` button. Click the `Add new site` you can see three options you click on deploy manually.
  3. Drag & drop your build folder there. Sometimes wait to deploy your site.
